# Allo Inventory Reservation System

A full-stack inventory reservation and order fulfillment platform built as part of the Allo Engineering take-home assessment.

The application simulates how modern ecommerce and D2C systems temporarily reserve inventory during checkout to prevent overselling while still ensuring abandoned or expired reservations return inventory back to available stock.

The project focuses heavily on:
- concurrency safety
- transactional consistency
- inventory correctness
- realistic backend architecture
- practical engineering tradeoffs
